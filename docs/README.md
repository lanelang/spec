# Specification Documentation

This directory contains language-owned design material.

- `adr/`: design records for source language semantics, type system rules,
  contextual resolution, pattern matching, and Buslane's specified role.
- `contexts/`: glossary slices relevant to the language specification.
- `existential-types.md`: the design note that was promoted into the
  specification.

Implementation-specific notes belong in the implementing repository.
